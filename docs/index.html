Console UNO

A classic card game, reimagined for your terminal!

About the Game

Console UNO is a command-line interface (CLI) implementation of the popular UNO card game. Play against AI opponents or with friends in local multiplayer mode. Experience the classic game with a retro console feel!

This project aims to provide a fun and accessible way to play UNO, focusing on core gameplay and a clean, text-based interface.

Features

  • Single-player mode against AI (1v1, 1v1v1, 1v1v1v1)
  • Local multiplayer mode (2-4 players)
  • Configurable settings for sound and animations
  • Cross-platform sound effects and music
  • Dynamic UI centering for a polished look
  • Self-contained executable for easy distribution

How to Play

Menu Navigation:

  • Numbers (1-9): Navigate menus

In-Game Commands:

  • Numbers (1-9): Play corresponding card
  • D: Draw a card
  • Q: Quit game

Settings Controls:

  • +: Increase volume
  • -: Decrease volume
  • Enter: Confirm volume
